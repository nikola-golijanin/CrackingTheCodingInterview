# Cracking the Coding Interview Solutions

This repository contains .NET Core solutions for coding challenges from the book "Cracking the Coding Interview" (CtCI). Each solution is structured to be easily run and debugged using Visual Studio Code or any compatible IDE.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- .NET Core SDK (version 8.0 or later)
- Visual Studio Code or any preferred IDE that supports .NET Core development

### Installing

1. Clone the repository to your local machine.
2. Open the project in Visual Studio Code or your preferred IDE.
3. Run the projct by running: dotnet run -- [Chapter].[Question]
4. Chapter will be in format "ChapterNN" and Question "QN" where N is number
example: 
```sh
    dotnet run -- Chapter01.Q2
```
5. If you are running project in VSCode then setup cli args in launch.json file
